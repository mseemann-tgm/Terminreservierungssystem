## Umsetzung
Die Applikationen sind IntelliJ Maven Projekte.
### Client
Benutzte Technologien:
* Java-Swing
* Jax-rs Client API
Execution:
```
mvn package
mvn exec:java
```
### Server
Benutzte Technologien:
* Jax-rs Server API
* HQL Querries
### Data
Benutzte Technologien:
* JPA Hibernate
Deployment:
```
mvn package
```
